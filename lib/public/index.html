Fish API

An App that provides the user with the tools to make the fishing experience better. Manages all aspects of the fishing trip and provides the user with information about the fish species and fishing regulations.:
Route Method Description Example 
/fishes
 GET Gets a list of all fish detail and ids List of fishes 
/fishes/:id
 GET Returns the detail about the fish matching the id in the parameter Get Fish 1 details 
/fishes/:id
 PUT Creates an update for an existing fish. Update Fish 3 
/fishes/
 POST Creates a new fish entry. Post a new Fish 
/fishes/
 GET Returns the fish and its regulations. (Protected routes for admin users ONLY) Get fish and regulations
Route Method Description Example 
/regulations
 GET Gets a list of all fish regulations and ids List of regulations 
/regulations/:id
 GET Returns the regulations about the fish matching the id in the parameter. (Protected routes for admin users ONLY) Get Fish 1 regulations 
/regulations/:id
 PUT Creates an update for an existing fishes regulation. (Protected routes for admin users ONLY) Update Fish 1 regulations
